# Tic-Tac-Toe Game

A classic Tic-Tac-Toe game built with vanilla HTML, CSS, and JavaScript. Play against another player in this interactive web-based implementation of the timeless board game.

## Features

- **Two-Player Gameplay**: Play as Player X and Player O
- **Turn Indicator**: Clear display of whose turn it is
- **Win Detection**: Automatically detects and announces winning combinations
- **Draw Detection**: Recognizes when the game ends in a draw
- **Game Reset**: Easily reset the board to start a new game
- **Responsive Design**: Clean, centered UI that works on different screen sizes

## How to Play

1. Open `index.html` in your web browser
2. The game starts with Player X's turn
3. Click on any empty cell to place your mark
4. Players alternate turns between X and O
5. Get three in a row (horizontally, vertically, or diagonally) to win
6. Click the "Reset Game" button to start over

## Project Structure

- `index.html` - Main HTML structure with game board and UI elements
- `style.css` - Styling for the game board, cells, and overall layout
- `script.js` - Game logic, turn management, and win/draw detection
